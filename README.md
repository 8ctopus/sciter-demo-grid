# sciter grid demo app

This is a [sciter](https://sciter.com/) js grids demo app.

## get started

- git clone the repository
- run `install.bat` to download the latest sciter binaries and library
- start `scapp.exe`
- to refresh the app after changes to the html/css click `F5`
- to view the other grids `scapp.exe scapp-01.html`

## debug app
- start `inspector.exe`
- inside the `scapp.exe` window click `CTRL + SHIFT + I` to connect to the inspector
- inside the `scapp.exe` window click `CTRL + SHIFT + left click` to inspect element in the inspector
